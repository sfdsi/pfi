La balise bouton (button) définit un bouton cliquable. Dans le code css on peut y attribuer une classe pour appliquer des styles différents. On peut y appliquer une pseudo-classe telle que ":hover" pour changer son apparence lorsque la souris passe au-dessus, ":active" lorsque le bouton est "cliqué" ou ":checked" qui change l'apparence de ce dernier une fois que sont lien a été visité. Cette type de balise est surtout pratique si l'utilisateur cherche à partager leur page d'Instagram, Facebook, Youtube, etc. sur la page Web. Pour ce faire, on peut mettre l'image du logo de la page externe et le lien en relation avec.
La balise "input" permet entre autre de créer un "checkbox". Ce checkbox permet à l'utilisateur de choisir une option ou plusieurs options d'une liste quelconque ou même d'un formulaire. Pour pouvoir l'utiliser, il faut lui attribuer un type et une valeur qui est envoyer au server une fois que le formulaire est soumise. On peut y appliquer des styles telles qu'un couleur de fond, une bordure ou encore un box-shadow.
La propriété "rotate" définit une valeur pour la rotation d'un élément dans le sens des aiguilles d'une montre autour de l'axe Z. Pour faire pivoter un élément autour de l'axe X ou Y ou d'une autre manière, cette valeur doit être définie. Cet élément est souvent utilisé pour créer l'effet de chargement d'une page ou dans divers autres situation qui demande un effet visuel dynamique. Nous pouvons y appliquer divers css tel que "transorm", "transition" ainsi que "animation".
Le menu déroulant est un menu de navigation qui apparait lorsque la souris passe au-dessus. C'est surtout pratique lorsque l'écran utilisé par l'utilisateur est d'une plus petite taille. On peut y appliquer des transitions ou des animations. De plus, il existe des variations du menu déroulant tel que "click dropdown", "multilevel dropdown" et "mega dropdown". En y appliquant les styles et pseudo-classes appropriés, nous pouvons créer un menu intéractif et dynamique à l'utilisateur.
Le pseudo-élément "::selection" est principalement utilisé pour modifier un texte selectionné par l'utilisateur. Il est possible d'appliquer une couleur différente au texte, de modifier la taille ou même d'ajouter une couleur de fond différent pour faire ressortir le tout.
